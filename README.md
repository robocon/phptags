phptag
======
Ref. http://www.php-fig.org/psr/psr-1/

Set php tags from &lt;? to &lt;?php

!!! IMPORTANT !!!
PLEASE BACKUP BEFORE YOU RUN THIS SCRIPT

Supports the following error tags
<?Test
<? Test
<?
<?
<?if()
<?Blog

อัพเดท php tags จาก &lt;? เป็น &lt;?php
